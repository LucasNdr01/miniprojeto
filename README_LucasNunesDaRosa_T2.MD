 **Reflexão sobre ETL**

O processo de ETL é importante para organizar os dados antes da análise.

.Extract: extração dos dados.
.Transform: limpeza e tratamento.
.Load: carregamento dos dados prontos para análise.

Sem esse processo os dados podem gerar análises erradas.

 **Reflexão sobre qualidade de dados**

Durante o projeto foi possível perceber que dados com erros, valores vazios ou duplicados atrapalham bastante as análises.
Por isso, a limpeza dos dados é uma etapa importante para deixar as informações mais organizadas.
